# devbana.github.io

Personal portfolio website for Akash S Devbanshi, built as a single-page static site.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

- `index.html`: Main application file (markup, styling, and behavior).

## Run Locally

1. Clone the repository.
2. Open `index.html` directly in your browser.

Optional (recommended) local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This repository is suitable for GitHub Pages deployment as a static site.
